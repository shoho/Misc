# HOT A HOTEL — Design System

> 熱をつれて、旅にでる。

薪火・サウナ・湯を滞在の中心に据えたホテルブランド(新規提案)のデザインシステム。

| ファイル | 内容 |
|---|---|
| [`tokens.css`](./tokens.css) | 全トークンの CSS カスタムプロパティ(ライト/ダーク両テーマ、4段構成) |
| [`design-system.md`](./design-system.md) | トークンの意図・タイポグラフィ・コンポーネント基本形・Do/Don't |
| [`index.html`](./index.html) | スタイルガイド(ブラウザでそのまま開ける。テーマトグル付き) |

コンセプトは **火と水の対比** — 熾火の朱 `#C63A1B` を 1 画面 1 主役に、湯けむりの青鈍 `#3E6B7A` を対抗色に。面は炭×白晒のニュートラルで静かに保つ。

`antigravity-design` の `/design` 系ワークフローから使う場合は、このディレクトリの `tokens.css` と `design-system.md` を既存デザインシステムとして参照させる。
